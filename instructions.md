# Reinforcement Learning Project Instructions

In this assignment you will solve the Taxi Gymnasium problem. You will solve it using: 

- Table-based Q-learning
- DQN (use a shallow network and/or regularization)


Optionally, you can apply the learnings to solve the Pong problem as well. For that, you can use either the original rgb-codification or the 128-bytes codification of the space.

Evaluation will be based on:

- Quantity, depth and quality of the work
- Results interpretation and analysis of the impact of the different hyperparameters / algorithms
- DQN vs Table comparison
- Optionally, the resolution of the Pong problem

Links:

- [Taxi Gymnasium](https://gymnasium.farama.org/environments/toy_text/taxi/)
- [Pong Gymnasium](https://ale.farama.org/environments/pong/)
